# Weather Data Management System

This project is dedicated to managing and analyzing weather data for three key locations in Germany: Mannheim, Berlin, and Zugspitze. The focus is on providing accurate temperature and air pressure data for these regions, with a specific emphasis on data collected from the year 2024.

## Project Overview
The Weather Data Management System aims to aggregate and analyze weather data, focusing on:
- **Temperature**: Tracking high and low temperatures throughout the year.
- **Air Pressure**: Monitoring air pressure fluctuations and trends.

By concentrating on Mannheim, Berlin, and Zugspitze, this application will serve as an essential tool for researchers, meteorologists, and enthusiasts interested in weather patterns and climate analysis.

## Data Sources
Data for this application has been sourced from reputable weather databases, ensuring accuracy and reliability. All data points are validated for consistency.

## Installation
To install the project, clone the repository and use the following command:

```bash
npm install
```

## Usage
To start the application, run:

```bash
npm start
```

This will initialize the weather data management system and allow you to interact with the data for Mannheim, Berlin, and Zugspitze.
